#Grudge-Box
>An app for storing a list of all the people who have wronged you.

###View
[Grudge Box](https://lk-grudge-box.herokuapp.com/)

###Built with:
* express
* node
* JavaScript
* jQuery
* PostgreSQL
* HTML/CSS

###Installation
####Install
'npm install'

####Run
'npm start'

####Test
'npm run test'
